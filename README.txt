== Description ==

rooftop-network-settings adds a GTM form to the network admin panel and inserts it into site admin pages.

Track progress, raise issues and contribute at http://github.com/rooftopcms/rooftop-network-settings

== Installation ==

rooftop-network-settings is a Composer plugin, so you can include it in your Composer.json.

Otherwise you can install manually:

1. Upload the `rooftop-network-settings` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There is no step 3 :-)

== Frequently Asked Questions ==

= Can this be used without Rooftop CMS? =

Yes, it's a Wordpress plugin you're welcome to use outside the context of Rooftop CMS. We haven't tested it, though.
